## 🎮 Gaming Performance Module v3.0 - Redmi Note 10S

Optimasi gaming untuk Redmi Note 10S (Helio G95) | Magisk/KernelSU Module

---

### ✨ Fitur
- CPU/GPU Performance Mode
- Auto Game Detection
- RAM Cleaner
- FPS Monitor
- Ping Stabilizer
- Terminal Game Booster

### 🎮 Game Supported
MLBB, PUBG, CODM, Genshin Impact, Free Fire, Wild Rift, dll

---

### 📥 Cara Install
1. Download zip dibawah
2. Install via Magisk/KernelSU
3. Reboot

### 🕹️ Cara Pakai
su -c gaming_tweaks

Menu Options:
- Optimize running games
- Extreme performance mode
- Balanced mode
- Clear RAM & cache
- FPS monitor
- Show device info

- Pakai cooling fan buat gaming lama
- Jangan sambil charge
